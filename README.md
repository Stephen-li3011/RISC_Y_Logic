# Summary
This application is a calendar tool students can use to organize their schedule. It allowes students to upload preexisting schedules to the calendar, manually add indiviual events, and recommend events based on the student's interest and time availability. This application was made to help students, like ourselves, navigate their busy lives and live up to their academic potential.
